브라우저 콘솔을 이용하여 로그 찍기

Monday, December 30 2013 4:21 AM

종종 Flash Debugger Player가 설치되지 못하는 환경에서 로그를 찍어봐야 할 때가 있다.

이럴때는 브라우저 콘솔을 사용하면 편하다.

private function traceLog( log:Object ):void
{
    ExternalInterface.call( "console.log", log );
}

이런 식으로 함수를 두고 사용하는 것.

이 방법을 사용할때 단점은 IE9 이전 버전에서는 console.log 를 지원하지 않는다는 것인데.

iOS App Lifecycle

Sunday, December 8 2013 9:30 AM

  • Not Running

    application이 아직 시작되기 전 또는 OS에 의해서 종료되었거나 크래쉬가 난 경우의 상태.

  • Inactive

    application이 foreground에서 실행중이지만 이벤트를 받지는 않는 상태.

    코드는 실행 가능.

    전화 / 문자가 왔을 경우, 기기가 잠금 상태 또는 다른 window가 overlay 되어 있을 때.

  • Active

    application이 foreground에서 실행중이며 이벤트를 받는 상태.

    사용자가 application을 사용중인 일반적인 상태.

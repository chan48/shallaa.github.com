개체가 'SetReturnValue' 속성이나 메서드를 지원하지 않습니다.

Tweet

**important 현상 ** 웹페이지에 삽입된 swf에서 ExternalInterface를 사용하는 경우 에러가 발생한다.

  • 환경 : Windows8.1 / IE11
  • 에러 메세지 : 개체가 'SetReturnValue' 속성이나 메서드를 지원하지 않습니다.

**inverse 원인 ** swf를 웹페이지에 삽입하는 코드가 object 태그안에 embed 태그로 작성되어 있고,

object, embed 태그의 id, name 속성이 중복되는 경우에 발생한다.

또는 object 태그의 id, name 속성이 없거나 embed 태그의 id, name 속성이 없는 경우.

예)

<object name="flahsObject" id="flahsObject" classid="clsid:d27cdb6e-ae6d-11cf-96b8-444553540000">
    <param name="movie" value="flahsObject.swf" />
    <embed name="flahsObject" id="flahsObject" 
        pluginspage="http://www.adobe.com/go/getflashplayer" src="flahsObject.swf" 
        type="application/x-shockwave-flash">
    </embed>
</object>

**info 해결 **

  • object, embed 둘 중 하나만 사용을 한다.
  • object, embed 태그의 id, name 을 각기 다르게 작성한다.

**warning 참고 ** IE에서는 object 태그를 사용해야한다. Internet Explorer 10 설치 후 Flash 객체의 메소드/속성 접근 오류가 발생한다면?

하.. 정말 IE11은 여러모로 피곤하네요. ㅎ

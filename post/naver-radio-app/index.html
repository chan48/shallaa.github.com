예전에 만들었던 네이버 라디오 어플을 GitHub로 옮겼다.

별 기능은 없고, 단순히 HTML 컴포넌트에 네이버 라디오 웹페이지를 넣고 트레이에서 실행이 가능하도록 처리를 했다.

네이버 라디오 웹페이지를 브라우저에 항상 열어두고 듣는 것보다 조금 나은 정도. ㅎ

처음 네이버 라디오가 나왔을때 우왕~ 하면서 만들었었는데.. 지금도 내 나름은 잘 쓰고 있는 편.

네이버 라디오를 이용하기 위해서는 이용권이랑 로그인이 필요한데 이용권은 사용중인게 있고, 로그인은 단순 무식하게 처리를 했다.

네이버 로그인 창을 HTML 컴포넌트에서 보여주고

Event.LOCATION_CHANGE 이벤트를 통해 HTML 컴포넌트의 location이 바뀔 때마다

location을 체크한 후에 로그인 성공창으로 이동한 경우에는 location을 네이버 라디오 창으로 바로 이동하도록 처리했다.

말로 하면 복잡한 느낌인데 코드로 보면 단순하다.

그리고 트레이의 context menu에서 몇몇 기능들( 다음곡, 좋아요, 싫어요 )을 바로 사용할 수 있도록 처리를 했다.

이 부분은 HTML 컴포넌트에서 그리고 있는 웹페이지의 window에 접근할 수 있다는 점을 이용했다.

window에 접근할 수 있으니까 당연히 window.document에도 얼마든지 접근이 가능하다.

이것도 방법이 좀 단순 무식하지만ㅎ 트레이의 context menu가 선택되면 웹페이지의 해당 기능을 하는 버튼에 접근 후 클릭 이벤트를 생성해서 dispatch 해버렸다.

대충 이런 식.
